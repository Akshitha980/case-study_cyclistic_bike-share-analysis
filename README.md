
# Cyclistic Bike Share Case Study

## 📌 Project Overview
This project is part of the Google Data Analytics Professional Certificate. 
The objective is to analyze 12 months of Cyclistic bike-share data to understand 
behavioral differences between casual riders and annual members.

## 🎯 Business Task
Identify usage patterns and provide data-driven recommendations to increase 
annual membership conversions.

## 🛠 Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Kaggle

## 🔎 Key Analysis Performed
- Data cleaning and transformation
- Removal of invalid and outlier ride durations
- Exploratory Data Analysis (EDA)
- Comparison of ride duration by user type
- Analysis of ride trends by month, weekday, and hour

## 📊 Key Insights
- Casual riders tend to take longer rides.
- Annual members use bikes more consistently.
- Usage patterns differ across weekdays and weekends.

## 🚀 Business Recommendations
- Target casual riders during peak leisure hours.
- Promote membership cost benefits to frequent riders.
- Launch seasonal marketing campaigns during high-demand months.
